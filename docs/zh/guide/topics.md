# 主题与标签

## 主题

主题是用于组织录制会话的项目级容器。

### 创建主题

1. 在导航栏中打开 **主题** 标签页
2. 点击 **创建主题**
3. 输入名称，选择一个 emoji 图标，可选添加描述

### 录制到主题中

两种方式将录制关联到主题：

- **从主题卡片** — 在主题卡片上点击 **新建录制**，打开 Live 页面并预选该主题
- **从 Live 页面** — 点击录制控件上方的 **选择主题** 链接并选择一个主题

选定的主题以标记形式显示在录制按钮上方，录制会自动关联。

### 管理主题中的会话

- 主题内的会话在默认历史记录列表中隐藏，但全局搜索仍可找到
- 在复盘工作台的 **概览** 标签页中将现有会话移入或移出主题
- 删除主题会移除关联关系但不会删除会话

## 标签

标签为会话提供轻量级的交叉标记。

### 使用标签

- 用名称和颜色（来自预设 Tailwind 调色板）创建标签
- 在复盘工作台概览标签页中为会话分配标签
- 在历史记录视图中按标签过滤会话
- AI 可在后处理中建议标签 — 一键应用

### 预设颜色

标签使用一组预定义的颜色组合，包含 `name`、`bg`、`text` 和 `border` 类，确保在明暗主题下的一致视觉效果。

## 备份与导入

- **导出** 将所有会话、标签和设置（不含主题）保存为 JSON 文件
- **导入** 可以 **覆盖** 所有数据或 **合并**（仅添加新会话和标签）
- 主题存储在 localStorage 中，不包含在备份文件中
