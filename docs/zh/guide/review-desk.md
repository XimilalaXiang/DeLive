# AI 复盘工作台

复盘工作台是用于探索已完成转录会话的全页工作区。从历史记录视图打开，提供四个标签页和动画滑动标签栏。

## 概览标签页

使用 OpenAI 兼容的 chat completions 端点从转录生成 AI 简报。

**输出包括：**
- **摘要** — 会话的简洁概述
- **行动项** — 提取的任务和后续步骤
- **关键词** — 讨论中的关键术语
- **章节** — 带可选时间戳的逻辑分段
- **标题建议** — AI 生成的会话标题
- **标签建议** — 用于组织的相关标签

一键 **应用** 按钮可直接采纳建议的标题和标签。

### 配置

AI 功能需要在 **设置 > 通用 > AI 后处理** 中配置 OpenAI 兼容端点：

- **Base URL** — 默认 `http://127.0.0.1:11434/v1`（Ollama）
- **模型** — 用于生成的模型
- **API Key** — 可选，取决于端点
- **提示语言** — `zh`（中文）或 `en`（英文）

## 转录标签页

显示完整转录：

- 左侧沟栏中的时间戳分段
- 彩色说话人标记（支持说话人分离时）
- 连续同说话人合并
- 单个分段悬停高亮
- 可选翻译文本块

**导出格式：** TXT、Markdown、SRT、VTT

## 对话标签页

关于转录的多线程 AI 对话：

- GFM Markdown 渲染，带语法高亮代码块
- 一键复制代码
- 用户/AI 头像
- 悬停操作（复制、重新生成）
- 动画思考点指示器
- 自动调整大小的文本输入框（Enter 发送）
- 悬浮滚动到底部按钮
- 按线程删除

每个问题发送时附带转录上下文和最多 4 轮先前问答以保持对话连续性。回答包含可选的 `citations`，引用转录中的特定片段。

## 思维导图标签页

从会话生成 Markmap 兼容的 Markdown 思维导图：

- 实时 Markdown 编辑器
- 交互式 Markmap 可视化
- 全屏 portal 模式
- 导出为 SVG 或 PNG

思维导图提示在可用时会利用 AI 摘要、行动项和关键词。

## 键盘导航

| 快捷键 | 动作 |
|--------|------|
| `←` / `→` | 切换标签页 |
| `Ctrl/Cmd + 1–4` | 跳转到特定标签页 |
| `Escape` | 关闭复盘工作台 |
