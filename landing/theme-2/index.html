Capabilities

Everything you need,
nothing you don't.

Six ASR engines, AI-powered review, floating captions — all running locally on your machine.

Six ASR Backends, One UI

Soniox, Volcengine, Groq, SiliconFlow, local OpenAI-compatible, and whisper.cpp. Three modes — streaming, batch, local.

Streaming Batch Local

Local-First & Secure

Sessions in IndexedDB, secrets in Electron safeStorage, context isolation. Your data never leaves your machine.

AI Review Desk

Summary, action items, keywords, chapters. Multi-thread chat with Markdown. Mind maps powered by Markmap.

Floating Caption Overlay

Always-on-top draggable subtitle window. Source, translated, and bilingual modes. Fully customizable.

Open API & MCP

REST API with 8 endpoints, WebSocket streaming, MCP server for Claude Desktop and Cursor.

Five Themes

Cyan, Violet, Rose, Green, Amber — each with light/dark mode. Sidebar nav and Command Palette.

WebDAV Cloud Backup

Nutstore, Nextcloud, Alist — any WebDAV server. Auto-create directories.
