Capabilities

Everything you need,
nothing you don't.

Six ASR engines, AI-powered review, floating captions — all running locally on your machine.

🎙️

Six ASR Backends, One UI

Soniox (real-time streaming with translation & diarization), Volcengine, Groq, SiliconFlow, local OpenAI-compatible, and local whisper.cpp. Three execution modes — real-time streaming, chunked batch, and local inference.

🔒

Local-First & Secure

Sessions in IndexedDB, secrets in Electron safeStorage, context isolation, and navigation guards. Your data never leaves your machine.

🧠

AI Review Desk

Overview with summary, action items, keywords, and chapters. Multi-thread chat with Markdown rendering. Mind maps powered by Markmap.

💬

Floating Caption Overlay

Always-on-top, draggable subtitle window with source, translated, and dual-line bilingual modes. Fully customizable font, color, shadow, and background. Perfect for meetings, lectures, and live events.

🌐

Open API & MCP

Local REST API with 8 endpoints, real-time WebSocket streaming, standalone MCP server for Claude Desktop and Cursor.

🎨

Five Themes

Cyan, Violet, Rose, Green, and Amber accent palettes — each with light and dark mode. Persistent sidebar navigation and Command Palette.

☁️

WebDAV Cloud Backup

Full WebDAV support — Nutstore, Nextcloud, Alist, and any standard WebDAV server. Auto-create directories.

Live Transcription

  • Multi-thread chat with transcript context
  • Interactive mind maps via Markmap
